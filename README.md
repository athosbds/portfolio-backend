## Overview

Portfólio profissional para desenvolvedor backend, destacando projetos reais desenvolvidos com **Python**, **FastAPI** e boas práticas de arquitetura. O site apresenta cases como API para MEIs, sistema BNPL e plataforma de ingressos, com foco em performance, segurança e escalabilidade.

---

## Features

- ✅ **Cards interativos** com projetos em destaque
- ✅ **Páginas detalhadas** para cada projeto (descrição completa, features, desafios técnicos)
- ✅ **Seção de stacks** organizada por categorias (Backend, Banco de Dados, Segurança, DevOps)
- ✅ **Links diretos** para GitHub (mesmo em repositórios privados)
- ✅ **Design responsivo** com suporte a tema claro/escuro
- ✅ **Animações suaves** e hover effects
